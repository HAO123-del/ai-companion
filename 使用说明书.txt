# AI 伴侣 使用说明书

## 📱 访问地址

**应用网址**: https://ai-companion-1.onrender.com

用手机或电脑浏览器打开即可使用。

---

## 🚀 快速开始

### 第一步：获取 MiniMax API 密钥

1. 访问 MiniMax 开放平台：https://platform.minimaxi.com
2. 注册/登录账号
3. 进入控制台，找到「API Keys」
4. 创建新的 API Key，复制保存
5. 同时记下你的 Group ID（在账户信息中）

### 第二步：配置 API

1. 打开应用 https://ai-companion-1.onrender.com
2. 点击右下角「设置」图标
3. 填入你的 MiniMax API Key 和 Group ID
4. 点击保存

### 第三步：创建 AI 伴侣

1. 回到首页，点击「创建伴侣」
2. 输入伴侣名字（如：小爱）
3. 设置性格描述（如：温柔体贴、幽默风趣）
4. 选择头像颜色和样式
5. 点击创建

## 🎯 功能介绍

### 💬 文字聊天
- 点击伴侣头像进入聊天
- 在底部输入框输入消息
- AI 会根据设定的性格回复你

### 📞 语音通话
- 在聊天界面点击「通话」按钮
- 允许麦克风权限
- 说话后 AI 会语音回复
- 支持克隆声音（需录制10秒以上清晰语音）

### 🎤 声音克隆
1. 进入「伴侣管理」页面
2. 点击「克隆声音」
3. 录制10秒以上清晰语音
4. 等待处理完成
5. 之后通话就会使用克隆的声音

### 📔 日记功能
- 记录每日心情和想法
- 查看心情统计和趋势
- 支持标签分类

### 🎮 小游戏
- 成语接龙
- 知识问答
- 和 AI 伴侣一起玩

### 📚 有声书
- 选择书籍
- AI 为你朗读
- 支持章节选择

### 🎵 音乐
- 播放背景音乐
- 放松心情

### ⏰ 提醒
- 设置定时提醒
- AI 会按时提醒你

### 🧠 记忆管理
- 查看 AI 记住的信息
- 添加/删除记忆
- 让 AI 更了解你

## ❓ 常见问题

### Q: 首次打开很慢？
A: 免费服务器会休眠，首次访问需等待30-60秒唤醒，之后就正常了。

### Q: 语音通话没声音？
A: 检查是否允许了麦克风权限，确保 API Key 配置正确。

### Q: 克隆声音失败？
A: 需要录制10秒以上清晰语音，避免背景噪音。

### Q: 数据会丢失吗？
A: 数据存储在服务器，但免费版可能不稳定，重要数据建议备份。

## 💡 使用技巧

1. **性格设定越详细，AI 回复越有个性**
   - 好的例子：「温柔体贴的女朋友，喜欢撒娇，说话带点可爱的语气词」
   - 不好的例子：「女朋友」

2. **语音通话时说话清晰**
   - 等 AI 说完再说话
   - 避免嘈杂环境

3. **多使用记忆功能**
   - 告诉 AI 你的喜好
   - AI 会记住并在聊天中体现
